Gizlilik Politikası

Web sitesi kullanımı sırasında toplanan kişisel veriler, kullanım amaçları, aktarım esasları ve kullanıcı hakları.

Gizlilik Sözleşmesi

Yürürlük Tarihi: 21.06.25
Web Sitesi: www.kalemekanik.com
Veri Sorumlusu: Kalemekanik

1. Giriş

Kalemekanik olarak, ziyaretçilerimizin ve kullanıcılarımızın gizliliğine önem veriyor; kişisel verilerinizin güvenliğini sağlamak için yasal düzenlemelere ve sektörel standartlara uygun hareket ediyoruz. Bu gizlilik politikası, web sitemizi kullanırken hangi bilgileri topladığımızı, nasıl kullandığımızı ve koruduğumuzu açıklamaktadır.

2. Toplanan Bilgiler

Web sitemizi ziyaret ettiğinizde veya hizmetlerimizden faydalandığınızda aşağıdaki türde kişisel veriler toplanabilir:

  • Kimlik Bilgileri: Ad, soyad, kullanıcı adı
  • İletişim Bilgileri: E-posta adresi, telefon numarası
  • Kullanım Verileri: IP adresi, tarayıcı bilgileri, çerez verileri
  • Konum Bilgileri: IP adresine dayalı yaklaşık konum verileri
  • Hizmet ve Sipariş Bilgileri: Formlar veya alışveriş işlemleri üzerinden sağlanan bilgiler

3. Verilerin Toplanma Yöntemi

Kişisel veriler;

  • Web sitemizdeki formların doldurulması,
  • E-posta veya telefon yoluyla bizimle iletişime geçilmesi,
  • Çerezler ve benzeri teknolojiler yoluyla, otomatik veya manuel yollarla toplanabilir.

4. Kişisel Verilerin Kullanım Amaçları

Topladığımız verileri aşağıdaki amaçlarla kullanabiliriz:

  • Size daha iyi hizmet sunabilmek
  • Taleplerinizi yanıtlamak
  • Teknik sorunları analiz edip çözmek
  • Kullanıcı deneyimini iyileştirmek
  • Mevzuata uygun şekilde kayıt ve belge düzenlemek
  • Pazarlama faaliyetleri (açık rıza alınması durumunda)

5. Üçüncü Taraflara Aktarım

Kişisel verileriniz, aşağıdaki durumlar haricinde üçüncü kişilerle paylaşılmaz:

  • Yasal yükümlülüklerin yerine getirilmesi
  • Hizmet aldığımız üçüncü taraf altyapı veya analiz sağlayıcıları (ör. hosting, kargo, ödeme altyapısı, e-posta servisi)
  • Açık rızanızın bulunması
  • Mahkeme, kolluk kuvvetleri veya resmi kurum talepleri

6. Çerezler (Cookies)

Web sitemizde kullanıcı deneyimini geliştirmek, trafik analizleri yapmak ve hedefli içerik sunmak amacıyla çerezler kullanılmaktadır. Tarayıcı ayarlarınızdan çerezleri reddedebilir veya silebilirsiniz. Ancak bu durumda bazı hizmetlere erişiminiz kısıtlanabilir.

7. Kişisel Verilerin Korunması

Verileriniz;

  • Şifreleme, firewall ve antivirüs sistemleri,
  • Yetkisiz erişime karşı rol bazlı kısıtlamalar,
  • SSL sertifikası gibi güvenlik teknolojileriyle korunmaktadır.

8. Haklarınız

6698 Sayılı KVKK uyarınca, aşağıdaki haklara sahipsiniz:

  • Hangi verilerin işlendiğini öğrenme
  • Verilerinizin düzeltilmesini talep etme
  • Silinmesini veya anonim hale getirilmesini isteme
  • İşlemenin durdurulmasını ya da sınırlandırılmasını talep etme
  • Açık rızanızı geri çekme
  • Kurula şikâyet hakkı

Haklarınızı kullanmak için info@kalemekanik.com adresine e-posta gönderebilirsiniz.

9. Değişiklikler

Bu gizlilik politikası zaman zaman güncellenebilir. Güncel haline web sitemiz üzerinden ulaşabilirsiniz.
